## 概要
このじゃんけんサイトでは下記の機能が利用できます。  
- 名前の入力  
- アザラシとのじゃんけん  

## 遊び方
1. 名前を入力します。（入力しない場合「じぶん」と表示されます）  
2. 下部のグー・チョキ・パーのうち、一つを選んでクリックします。  
3. アザラシ（あいて）はランダムでグー・チョキ・パーのいずれかを出します。
4. 勝負の結果が表示されます。
